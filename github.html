¿Qué es GitHub?

GigHub es una herramienta gratuita que aloja un servicio de control de versiones y gestión de proyectos llamado Git, el cual nos permite almacenar a modo de repositorio, todo el código o recursos de un proyecto, permitiéndonos guardar estados y llevar un registro de cambios. Github resulta en una herramienta muy útil al momento de trabajar colaborativamente en un proyecto, pues nos permite gestionar los avances y evitar duplicidad o perdida de trabajo.

GitHub es un proyecto desarrollado en 2008 por Chris Wanstrath , PJ Hyett , Tom Preston-Werner y Scott Chacon, para febrero del año 2009 esta plataforma ya contaba con 46 mil repositorios públicos y para julio del mismo año ya superaba los 135 mil repositorios. Al día de hoy esta es una de las herramientas más usadas en el desarrollo de software y si bien ahora hay más opciones GitHub sigue encabezando las listas de popularidad.
